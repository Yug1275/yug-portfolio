# Yug Patel - Developer Portfolio

This is my personal **full-stack developer portfolio** built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.  
It showcases my **projects, skills, education, experience, and achievements** in a clean and modern interface.

The portfolio is designed to be **responsive, interactive, and dynamic**, with project data fetched from a backend API.

## ⚙️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Deployment
- Netlify (Frontend)
- Render (Backend)
- MongoDB Atlas (Database)
